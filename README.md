# Enciclopédia de Momentos Históricos do Futebol

Repositório colaborativo da turma de **Desenvolvimento Ágil de Software** (ISTEC).
Cada aluno documenta **um momento icónico** da história do futebol. No fim, teremos uma enciclopédia feita por toda a turma.

## Missão

Escolher um momento histórico e criar o **teu** ficheiro em `momentos/`, contando o que aconteceu e porque é que ficou para a história.
> Ideias: o Maracanaço (1950) · o Brasil de Pelé (1970) · a "mão de Deus" (1986) · Portugal campeão do Euro 2016 · a final de 2014 · o teu momento favorito!

## Como contribuir

1. Aceita o convite de colaborador.
2. Clona o repositório:
   ```bash
   git clone <url-deste-repositorio>
   cd momentos-futebol-istec
   ```
3. Cria o **teu** ficheiro `momentos/nome-apelido.md` (copia o `momentos/_MODELO.md`).
4. Escreve o teu momento em Markdown.
5. Guarda e envia:
   ```bash
   git pull
   git add momentos/nome-apelido.md
   git commit -m "Adiciona momento historico (<nome>)"
   git push
   ```

## Regras

- **Um ficheiro por pessoa** — mexe só no teu (`momentos/nome-apelido.md`).
- Nomes **sem espaços nem acentos**: `afonso-tavares.md`, não `Afonso Tavares.md`.
- **`pull` antes, `push` depois.**
- Um momento por pessoa — se já estiver escolhido, escolhe outro!
